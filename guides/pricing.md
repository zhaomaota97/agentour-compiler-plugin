# 积分定价指南

Agentour 平台唯一货币是**积分**。Agent 按次计费，消费者每次调用扣一次。

## 定价参考

| Agent 复杂度 | 建议价格 | 典型场景 |
|---|---|---|
| 简单查询/计算 | 5-10 积分 | 装修报价估算、条款风险打分 |
| 中等流程（多步工具） | 10-20 积分 | 出餐核查、酒店查房、出库核验 |
| 专业审核（领域知识密集） | 20-40 积分 | 处方审核、4S 维修审计 |

## 原则

- 定价反映 Agent 提供的**业务价值**，不是 LLM token 成本
- LLM 消耗由平台统一承担，开发者不需要关心
- `agentour.json` 的 `pricing.amount_cents` 字段填写积分数量（整数）
- 消费者在发现页看到明确的价格标签
