# Agent 图标指南

Agent 在市场中的图标用于快速识别。默认使用 emoji，也支持上传自定义图片。

## Emoji 图标（推荐）

直接使用单 emoji 作为图标，在 `berth.json` 中设置 `icon` 字段：

| 行业 | 推荐 emoji |
|---|---|
| 餐饮/食品 | 🍔 🍕 🍜 ☕ |
| 医疗/药学 | 💊 🏥 🩺 🧬 |
| 汽车/机械 | 🔧 🚗 🔩 ⚙️ |
| 酒店/旅游 | 🏨 🛏️ 🧳 ✈️ |
| 物业/建筑 | 🏢 🏗️ 🔨 📐 |
| 仓储/物流 | 📦 🚛 🏭 📋 |
| 法务/合同 | 📄 ⚖️ 📋 🔏 |
| 财务/审计 | 💰 📊 🧾 🔍 |
| 教育/培训 | 📚 ✏️ 🎓 🏫 |

## 自定义图片（高级）

1. 准备 256×256 的 PNG 或 SVG 文件
2. 放在 `payload/` 目录下（例如 `payload/icon.png`）
3. 在 `berth.json` 中设置 `icon` 为文件路径（例如 `icon: "icon.png"`）

## 原则

- 图标应直观传达 Agent 的行业/用途
- emoji 最简单，覆盖大部分场景
- 避免使用文字作为图标（缩放后不可读）
